# Muhammad Talha Fahad Portfolio

This folder contains a complete static portfolio website.

## Open Locally

Open `index.html` in a browser.

## Resume

The resume is attached at:

`assets/Hafiz_Talha_CV.pdf`

The "View Resume" button on the website opens this PDF in a modal.

## Deploy

You can upload this folder to Vercel, Netlify, GitHub Pages, or any static hosting service.

## Main Files

- `index.html` - page content and structure
- `style.css` - layout, colors, responsive design, and animations
- `script.js` - typing effect, modal, contact form, timeline, and interactions
- `3d_background.js` - Three.js star background
- `assets/` - resume and profile image
- `vendor/` - local Three.js and GSAP runtime files
